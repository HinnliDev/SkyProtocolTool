# Sky Protocol Tool (SPT)
[![English](https://img.shields.io/badge/lang-English-red)](./README.md)
[![License MIT](https://badgen.net/github/license/HinnliDev/SkyProtocolTool)](https://github.com/HinnliDev/SkyProtocolTool/blob/main/LICENSE)

Sky Protocol Tool (SPT) — это плагин для Android-версии Sky: Children of the Light, загружаемый через Canvas. Он предоставляет удобный интерфейс для анализа сетевого HTTP-трафика игры и ручной работы с запросами.
</div>

---

## Возможности

- Перехват и отображение HTTP-запросов и ответов
- Консоль с группировкой request/response
- Фильтрация и поиск по логам
- Генерация команд из выбранных запросов
- Ручная отправка HTTP-запросов
- Перехват команд через игровой чат
- Копирование данных

---

## Как работает

SPT подключается к внутренним HTTP-функциям игры и:
- перехватывает исходящие запросы
- анализирует входящие ответы
- отображает всё в UI

Также используются реальные шаблоны запросов для отправки собственных команд.

---

## Интерфейс

Основной интерфейс включает:
- консоль логов
- фильтры и поиск
- просмотр запросов
- поле для ручного ввода

Типы трафика:
- `C->S` / `SPT->S` — клиент → сервер
- `C<-S` / `SPT<-S` — сервер → клиент

---

## Ручные запросы

Формат:

```
/post /endpoint -c key:value
```

Примеры:

```
/post /account/info -c user:123
/post /service/ping -c enabled:true
/post /example -c {"key":"value"}
```

### Команды через чат

Если сообщение содержит команду `/post`, она перехватывается и выполняется
> В чат ничего не отправляется и другие игроки её не видят.
